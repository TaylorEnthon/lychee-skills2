# 错误码提示

| code | 用户提示 |
| --- | --- |
| `1000` | 服务暂时不可用，请稍后重试。 |
| `1001` | 参数不符合要求，请按返回信息检查文本、模式和引用数量。 |
| `1003` | 图片或参考音频无效，请检查格式和大小。 |
| `1100` | 请运行 `/lychee-set-key` 配置 API Key。 |
| `1400` | 文件上传失败，请重试。 |
| `1401` | 文件下载失败，请确认链接可访问。 |
| `1500` | 配音服务返回业务错误，请重试或简化文本。 |
| `1501` | 配音服务网络异常，请稍后重试。 |

CLI 会保留后端原始错误文本，便于定位具体字段。
