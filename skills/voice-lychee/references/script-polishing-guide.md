# 剧本润色指南（voice-lychee TTS 后端专用）

> 适用场景：用户给了一个简单剧本（甚至只是一段台词），想让 AI 配音出"有声剧"效果。
> 目标：把糙剧本润色成 TTS 后端友好的"表演+音效"剧本。

## 0. TTS 后端能做什么 / 不能做什么

### ✅ 能做
- **多音色分角色**（用 `--voice-ids` + `@音频N` 切分台词）
- **情感/语气控制**（在台词前用 prompt 描述声音情绪）
- **环境混响**（在开头 prompt 描述空间感）
- **拟音词嵌入**（TTS 会读出中文拟音字"咚/唰/咻/哗"作为音效替代）
- **基础背景音**（用 prompt 描述"低沉二胡全程铺垫"——TTS 可能生成近似音色，但效果有限）

### ❌ 不能做
- **真正的 BGM 合成**（TTS 不生成音乐）——只能用 prompt 描述"BGM 名称"作为风格提示
- **后期混音**（语音 + 音乐 + 音效同轨混合）——需要在外部工具里合成

## 1. 润色后的剧本结构

```
【场景设定】
角色1（音色 + 情感）：台词
[音效/拟音词]
角色2（音色 + 情感）：台词
[音效/拟音词]
...
【情绪轨迹备注】 ← 可选（写给 agent 看）
```

## 2. 五要素（缺一不可）

### 2.1 角色卡片（每个角色 1 行）

格式：`角色名（性别 + 年龄段 + 音色特征 + 情感轨迹）：`

| 维度 | 必填 | 示例 |
|---|---|---|
| 性别 | ✅ | 男/女 |
| 年龄段 | ✅ | 青年/中年/中老年/老年 |
| 音色特征 | ✅ | 清亮/尖柔/浑厚/干涩/沙哑/温润 |
| 情感轨迹 | 可选 | 前期清亮激动，中段哽咽颤抖，后期沙哑死寂 |

**示例**：
- 沈清晏（女主）：青年女声，前期清亮激动，中段哽咽颤抖，后期沙哑死寂
- 贵妃：青年女声，尖柔伪善，假意委屈
- 皇帝：中年男声，浑厚威严，冷漠疏离
- 太监：中老年男声，干涩低沉

### 2.2 表演提示（每段台词前 1 行）

格式：`角色名 + 动作描写 + 声音情绪：`

| 要素 | 示例 |
|---|---|
| 动作 | "望着桌上全套栽赃毒药" / "垂眸垂泪" / "浑身僵住" |
| 声音情绪 | "声音急促拔高" / "声线柔弱虚伪" / "浑厚嗓音冰冷无温" / "声音哽咽拔高" / "声音颤抖无力" / "嗓音沙哑死寂" |
| 对白标记 | 用冒号或引号引出："台词内容" |

**原则**：
- 用**动作描写**而不是直接说情绪（"声音哽咽"比"哭"更 TTS 友好）
- 描写具体**声音特征**（"急促拔高""颤抖无力""沙哑死寂"）而不是笼统词
- 表演提示要**贴在台词前面**（不是后面）

**示例**：
- 沈清晏望着桌上全套栽赃毒药，声音急促拔高、奋力辩解："陛下！此药绝非臣妾所有..."
- 贵妃垂眸垂泪，声线柔弱虚伪："妹妹怎可颠倒黑白..."
- 皇帝浑厚嗓音冰冷无温："证据确凿，不必多言。"

### 2.3 拟音词库（中文拟音字嵌入文本）

TTS 会读出**单字拟音词**作为音效——这是把"音效"塞进 TTS 的核心技巧。

| 类型 | 拟音字 | 触发场景 |
|---|---|---|
| 重物落地 | "咚" "嘭" "轰" | 玉簪摔落、瓷器碎裂、重物倒地 |
| 布料 | "唰" "沙" "窸窣" | 锦缎长裙拖地、衣袖拂动 |
| 脚步 | "咻" "踢踏" "踏踏" | 快步、慢步、慌乱小跑 |
| 陶瓷碎裂 | "哗" "哗啦" "噼啪" | 玉瓶碎裂、瓷碗打碎 |
| 金属 | "叮" "当" "嗡" | 玉饰相撞、铜铃声 |
| 风 | "呼" "呼呼" | 冷风、穿堂风 |
| 雷 | "轰隆" "闷雷" | 远处闷雷、雷声 |
| 门 | "吱呀" "嘎吱" | 宫门、房门开合 |
| 布料/身体 | "扑通" "啪嗒" | 跪地、摔倒 |
| 泪珠 | "啪嗒" "滴答" | 泪珠落衣襟 |

**使用方法**：拟音词**夹在台词之间或前后**，用方括号或单独成行：
- "台词间穿插锦缎裙摆大幅滑动 '唰'、宫鞋急促踩踏地砖、玉瓶被扫落在地碎裂哗啦"
- "二胡旋律骤然悲凉下沉，太监上前押人，布料摩擦声响响起"
- "沈清晏浑身僵住，短促一声轻 '啊'，泪珠重重砸落在衣襟"

### 2.4 背景音/BGM 提示（剧本开头）

TTS 不能合成真正的 BGM，但 prompt 描述的"低沉二胡全程铺垫"会让 TTS 调整输出风格。

**常见 BGM 描述模板**：
- 低沉二胡搭配淡合成器 pad 全程铺垫，氛围压抑悲戚
- 古筝轻拨配合清脆铃音，氛围清新雅致
- 紧张弦乐层层递进，氛围悬疑紧迫
- 优雅圆舞曲轻快节奏，氛围欢快明亮
- 寂静夜晚虫鸣蛙叫，氛围田园恬静

**位置**：剧本**最开头**写一行。

### 2.5 空间混响/转场（场景结束/转场）

**混响模板**：
- 大殿空间自带厚重空旷混响
- 小房间近距细腻混响，声音紧凑
- 户外开放空间无明显混响
- 山谷回音悠长

**转场技巧**：用"声音转场"+ 拟音词过渡：
- "殿门冷风 '呼' 转场"
- "场景转换，画面淡出"
- "蒙太奇，时间跳转"

## 3. 完整剧本模板

```
【场景：xxxxx】
【BGM：xxxxx】
【空间：xxxxx】

角色A（音色 + 情感）：台词
[拟音词 / 音效描写]

角色B（音色 + 情感）：台词
[拟音词 / 音效描写]

...

【情绪轨迹】
角色A 前期 → 中期 → 后期 的情绪变化
```

## 4. 润色步骤（agent 应该执行）

1. **识别角色 + 台词** —— 把用户给的简单对话拆成"谁说的" + "说了什么"
2. **补全角色卡片** —— 推断每个角色的性别、年龄段、音色特征；如果用户已指定就用用户的
3. **加表演提示** —— 每段台词前加"动作 + 声音情绪"描写
4. **加拟音词** —— 在场景转换、关键情绪点、动作发生时插入拟音字
5. **加 BGM + 混响** —— 剧本开头补"低沉二胡""大殿空旷混响"等场景描述
6. **加情绪轨迹** —— 如果有角色情感变化，写明"前期→中期→后期"轨迹
7. **整合输出** —— 上面所有内容拼成一个完整剧本

## 5. 完整润色前后对比

### 润色前（用户输入）：
```
沈清晏：求陛下明察！
贵妃：你胡说的。
皇帝：证据确凿。
沈清晏：我无话可说。
```

### 润色后：
```
【场景：皇宫大殿深夜，栽赃审讯】
【BGM：低沉二胡搭配淡合成器 pad 全程铺垫，氛围压抑悲戚】
【空间：大殿空间自带厚重空旷混响】

沈清晏（青年女声，前期清亮激动，中段哽咽颤抖）：声音急促拔高、奋力辩解："陛下！此药绝非臣妾所有，是贵妃刻意陷害，求陛下明察！"
[锦缎裙摆大幅滑动 "唰"、宫鞋急促踩踏地砖、玉瓶被扫落在地碎裂哗啦]

贵妃（中青年女声，尖柔伪善）：垂眸垂泪，声线柔弱虚伪："妹妹怎可颠倒黑白，人证物证俱在，何苦狡辩。"
[宫女布鞋整齐行走脚步声]

皇帝（中年男声，浑厚威严）：嗓音冰冷无温："证据确凿，不必多言。"
[宫人摔倒地砖闷响、玉饰相撞金属嗡鸣]

沈清晏：声音哽咽颤抖、无力挣扎："陛下，您从前信我…… 为何如今半句不听……"
[二胡旋律骤然悲凉下沉，太监上前押人，布料摩擦声响]

沈清晏：嗓音沙哑死寂，再无半分争执力气："原来再多真心，抵不过一场精心设计的圈套。我无话可说。"
[珠钗散落满地哗啦、膝盖重重跪地闷响、冷风呼啸 "呼" 转场]
```

## 6. 把润色后的剧本转成 synthesize.py 调用

润色后的剧本**不直接调 synthesize.py**——需要拆成多段 + 多 `--voice-ids`：

1. **识别所有 unique 角色 → 查音色池**（用 `--voice-names` 拿 ID）
2. **每段台词独立调一次 synthesize.py**（`--voice-ids` + `@音频N` 单角色）
3. **后期合并音频**（用 `ffmpeg` concat）——但**当前 skill 不做这一步**（超出范围）

**如果用户想一气呵成合成整段**（用 voices 模式 + `@音频N` 切角色）——把整段剧本拼成一个 `text_prompt`：
```
{{voice:1}}声音急促拔高、奋力辩解："陛下！此药绝非臣妾所有..."[拟音词]
{{voice:2}}声线柔弱虚伪："妹妹怎可颠倒黑白..."
{{voice:1}}声音哽咽颤抖："陛下，您从前信我..."
```

但**字符数限制 3000**——长剧本可能超长，需要分批合成再 concat。
